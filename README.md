# 🧬 DNA Info

My first bioinformatics project written in Python.

This project is being developed step by step while I learn Python. The goal is to create a small tool for DNA sequence analysis.

## Current features

- Input a DNA sequence
- Display the sequence
- Calculate sequence length

## Planned features

- Validate DNA sequences
- Count A, T, G and C nucleotides
- Calculate GC content
- Read FASTA files
- Reverse complement
- DNA → Protein translation

## Technologies

- Python 3
